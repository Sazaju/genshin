Version de développement

Ce site est la version de développement. Le code peut changer de manière incompatible. Des erreurs peuvent donc survenir à n'importe quel moment. Utilisez cette version à vos risques et périls.

Toutes les données affichées sont stockées localement (sur votre machine). Si vous rencontrez des problèmes, utilisez le bouton RAZ pour effacer toutes ces données et repartir d'une version propre. Utilisez le bouton de stockage pour voir ce qui est stocké sur votre navigateur dans la console. Si vous n'arrivez pas à régler votre problème, n'hésitez pas à passer par la page de contact.
